***********************
Отток клиентов банка
***********************
Описание проекта: Из "Бета-Банка" стали уходить клиенты. Предоставлены исторические данные о поведении клиентов и расторжении договоров.

Задача: построить модель для прогноза: уйдёт клиент в ближайшее время или нет.

Ход исследования:
1. Загрузка и подготовка данных.
2. Исследование баланса классов, обучение модели без учёта дисбаланса.
3. Улучшение качества модели, с учётом дисбаланса классов.
4. Выбор лучшей модели.
5. Оценка качества модели с помощью метрик: F1-мера (минимум 0,59) и AUC-ROC.
6. Тестирование модели на тестовой выборке.

Используемые модели:
LogisticRegression
DecisionTreeClassifier
RandomForestClassifier
CatBoostClassifier
